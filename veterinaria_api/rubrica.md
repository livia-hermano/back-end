# Rubrica de Avaliação: Desafio Veterinária API (MVC)

**Nível 1 - Insuficiente (0 - 25%):** Não conseguiu identificar os arquivos. Estrutura de pastas inexistente ou incorreta. Não encaixou os trechos.
**Nível 2 - Em Desenvolvimento (26% - 50%):** Criou as pastas, mas errou a nomenclatura. Inseriu trechos de forma errada. API não inicializa.
**Nível 3 - Proficiente (51% - 85%):** Recriou bem a estrutura. Conseguiu encaixar quase todos os trechos de código com sucesso. O CRUD funciona quase 100%.
**Nível 4 - Excelente (86% - 100%):** Estrutura impecável. Identificou todos os arquivos, resolveu o quebra-cabeça e a API roda lisa, validando todas as regras (idade, etc) perfeitamente.
